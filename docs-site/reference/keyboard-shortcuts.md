# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in HiveMind OS. On macOS, substitute `Cmd` for `Ctrl`.

## Chat

| Shortcut | Action |
|---|---|
| `Enter` | Send message |
| `Shift+Enter` | Insert newline |
| `Ctrl+C` | Stop the current agent |
| `Ctrl+/` | Open slash command palette |
| `Ctrl+K` | Open command palette |
| `Up` | Cycle through message history |
| `Escape` | Close autocomplete / cancel |

## Navigation

| Shortcut | Action |
|---|---|
| `Ctrl+N` | New chat session |
| `Ctrl+Tab` | Switch to next session |
| `Ctrl+Shift+Tab` | Switch to previous session |
| `Ctrl+B` | Toggle sidebar |
| `Ctrl+,` | Open settings |
| `Ctrl+P` | Quick open / search sessions |

## Bots

| Shortcut | Action |
|---|---|
| `Ctrl+Shift+B` | Open bots dashboard |
| `Ctrl+Shift+L` | Launch a new bot |
| `Ctrl+Shift+S` | Stop active bot |

## Editing

| Shortcut | Action |
|---|---|
| `Ctrl+Z` | Undo last edit |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+A` | Select all text in input |
| `Ctrl+Shift+C` | Copy last response |

::: tip
Press `Ctrl+K` to open the command palette and search for any action by name — useful when you forget a shortcut.
:::

::: warning
Some shortcuts may conflict with terminal emulator bindings. Check your terminal settings if a shortcut doesn't work as expected.
:::
