# @hivemind-os/connector-github-issues

Reference GitHub Issues connector plugin for Hivemind.

## What it does

- list and search issues
- create issues
- add comments
- run a background notification loop

## Development

```bash
npm install
npm run build
npm test
```

## Local linking

```bash
hivemind plugin link .
```

This package is the sample connector used by the in-app plugin registry.
